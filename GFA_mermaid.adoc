== Mermaid Preview Examples

We can preview Mermaid diagrams on GitHub not only in Markdown (GFM) but also in GitHub Flavored AsciiDoc (GFA).

[source,mermaid]
----
flowchart LR
  Sleep[Sleep] --> Wake{Awake?}
  Wake -->|No| Sleep
  Wake -->|Hungry| Snack[Get treat]
  Wake -->|Not in in Sun?| Move[Move to sun]
  Wake -->|Human is typing| Keyboard[Sleep on keyboard]
  Snack --> Sleep
  Move --> Sleep
  Keyboard --> Sleep
----
